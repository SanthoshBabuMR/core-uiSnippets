UI Snippets

**Dependencies**
* grunt-contrib-connect
* grunt-contrib-jshint
* grunt-jshint-stylish

**Launching server from grunt**
* Default/App Server ( [http://localhost:6234/](http://localhost:6234/) )
```bash
grunt appServer
```
* lint grunt module(s)
```bash
grunt lintGrunt
```
* lint application
```bash
grunt lintApp
```
* lint grunt and application module(s)
```bash
grunt lint
```
